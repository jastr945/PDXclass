Caesar cipher is said to have been used by Julius Caesar to communicate with his army. Caesar cipher is a way of encrypting simple text using a single number as a key. Each letter is "rotated" forward or backward by that key number. The ciphertext alphabet is derived from the plaintext alphabet by shifting each letter a certain number of spaces. Try it yourself!
